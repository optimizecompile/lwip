# 数据包管理 (PBUF)

## 概述

**pbuf** 是 lwIP 中用于管理网络数据包的核心数据结构。它提供了灵活的数据包存储和传递机制。

## 源文件

- `src/core/pbuf.c` - pbuf 实现
- `include/lwip/pbuf.h` - 头文件

## PBUF 结构

```c
struct pbuf {
    struct pbuf *next;      // 下一个 pbuf 链
    void *payload;          // 数据指针
    u16_t len;              // 当前 pbuf 长度
    u16_t tot_len;          // 链总长度
    u16_t type;             // pbuf 类型
    u8_t flags;             // 标志位
    u8_t ref;               // 引用计数
};
```

## PBUF 类型

| 类型 | 描述 | 用途 |
|------|------|------|
| `PBUF_RAM` | 连续内存块 | 发送/接收数据 |
| `PBUF_POOL` | 内存池分配 | 高性能场景 |
| `PBUF_ROM` | 只读数据 | 静态数据发送 |
| `PBUF_REF` | 引用外部数据 | 零拷贝场景 |

### RAM 类型

```c
// 分配 RAM pbuf
struct pbuf *p = pbuf_alloc(PBUF_TRANSPORT, TCP_MSS, PBUF_RAM);
```

### POOL 类型

```c
// 分配 POOL pbuf - 从内存池获取，速度快
struct pbuf *p = pbuf_alloc(PBUF_LINK, 128, PBUF_POOL);
```

## API 操作

### 分配

```c
#include "lwip/pbuf.h"

// 分配 pbuf
struct pbuf *pbuf_alloc(pbuf_layer layer, u16_t length, pbuf_type type);

// 释放 pbuf
void pbuf_free(struct pbuf *p);

// 引用计数增加
void pbuf_ref(struct pbuf *p);
```

### 数据操作

```c
// 复制 pbuf
struct pbuf *pbuf_copy(struct pbuf *p);

// 跳过指定字节
err_t pbuf_try_skip(struct pbuf *p, u16_t skip);

// 获取链总长度
u16_t pbuf_len = pbuf_copy_partial(p, buffer, len, offset);
```

### 链式操作

```c
// 添加到链尾
void pbuf_cat(struct pbuf *head, struct pbuf *tail);

// 克隆 pbuf 链
struct pbuf *pbuf_clone(PBUF_TO_CLONE, PBUF_FROM_CLONE);
```

## PBUF 层级

| 层级 | 描述 | 示例 |
|------|------|------|
| `PBUF_RAW` | 原始数据 | 链路层 |
| `PBUF_IP` | IP 层 | 网络层 |
| `PBUF_TRANSPORT` | 传输层 | TCP/UDP |
| `PBUF_APP` | 应用层 | 用户数据 |

## 内存布局

```
pbuf 链示例:

+-------+-------+-------+-------+
| pbuf  | pbuf  | pbuf  | pbuf  |
| ref=1 | ref=1 | ref=1 | ref=1 |
+-------+-------+-------+-------+
| payload | payload | payload |
+---------+---------+---------+
```

## 配置选项

```c
// lwipopts.h
#define PBUF_POOL_SIZE          256     // 池中 pbuf 数量
#define PBUF_LINK_HLEN          16      // 链路层头长度
#define PBUF_IP_HLEN            20      // IP 头长度
#define PBUF_TRANSPORT_HLEN     20      // 传输层头长度
```

## 最佳实践

> [!warning] 注意事项
> - 发送数据优先使用 `PBUF_POOL`
> - 大数据分片注意 `pbuf_clamp`
> - 及时释放不再使用的 pbuf
