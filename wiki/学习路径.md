# lwIP 学习路径

## 推荐学习顺序

建议按照以下顺序学习 lwIP，由浅入深，循序渐进。

## 第一阶段：基础入门

### 1. 了解 TCP/IP 基础

> [!note] 前置知识
> - 网络分层模型 (OSI 七层 / TCP/IP 四层)
> - IP、TCP、UDP 协议基本原理
> - Socket 编程概念

### 2. lwIP 概述

- [[lwIP 概述]] - 项目简介、架构、特性
- 阅读 `README` 和 `FEATURES` 文件

### 3. 建立开发环境

```bash
# 查看 contrib/ports/unix 获取 Unix 模拟环境
cd contrib/ports/unix

# 查看示例应用
cd contrib/examples/
```

## 第二阶段：核心机制

### 4. 内存管理

- [[内存管理]] - 堆内存与内存池
- 阅读 `src/core/mem.c` 和 `src/core/memp.c`

### 5. 数据包管理

- [[数据包管理]] - pbuf 结构与操作
- 阅读 `src/core/pbuf.c`

### 6. 网络接口

> [!tip] 关键概念
> - `netif` 结构
> - 网卡注册与初始化
> - 网卡驱动框架

**实践任务**: 实现一个简单的网卡驱动框架

## 第三阶段：协议实现

### 7. IP 协议

- 学习 `src/core/ipv4/ip4.c`
- 理解 IP 分片与重组

### 8. TCP 协议

- [[TCP 协议]] - 状态机、窗口管理
- 阅读 `src/core/tcp.c`

**实践任务**: 实现 TCP Echo 服务器

### 9. UDP 协议

- [[UDP 协议]] - 无连接通信
- 阅读 `src/core/udp.c`

**实践任务**: 实现 UDP Echo 服务器

### 10. 辅助协议

| 协议 | 文件 | 用途 |
|------|------|------|
| DHCP | `src/core/ipv4/dhcp.c` | 动态 IP 获取 |
| DNS | `src/core/dns.c` | 域名解析 |
| ICMP | `src/core/ipv4/icmp.c` | 网络诊断 |

## 第四阶段：API 层

### 11. Netconn API

- [[Netconn API]] - lwIP 原生 API
- 阅读 `src/api/api_msg.c`

### 12. BSD Socket API

- [[BSD Socket API]] - POSIX 兼容接口
- 阅读 `src/api/sockets.c`

**实践任务**: 将 Echo 服务器移植到 Socket API

## 第五阶段：应用层协议

### 13. HTTP 服务器

- 学习 `src/apps/http/httpd.c`
- 理解 CGI 和 SSI

**实践任务**: 搭建一个简单的 Web 服务器

### 14. MQTT 客户端

- 学习 `src/apps/mqtt/mqtt.c`
- 阅读 `contrib/examples/mqtt/`

### 15. SNMP 代理

- 学习 `src/apps/snmp/`
- 实现自定义 MIB

## 第六阶段：高级主题

### 16. IPv6 支持

- `src/core/ipv6/` 目录
- 双栈配置

### 17. PPP/PPPoS

- `src/netif/ppp/` 目录
- 串口网络连接

### 18. 性能优化

```c
// 关键配置参数
#define MEM_SIZE              // 堆大小
#define PBUF_POOL_SIZE       // pbuf 池
#define TCP_WND              // TCP 窗口
#define TCP_MSS              // MSS 大小
```

### 19. 线程模型

- NoSYS 模式 vs OS 模式
- tcpip_thread 和 netif_thread

## 实践项目建议

### 初级项目
1. TCP/UDP Echo 服务器
2. HTTP 静态文件服务器
3. DHCP 客户端

### 中级项目
1. MQTT 客户端连接云平台
2. Webserver + CGI 动态页面
3. DNS 客户端

### 高级项目
1. HTTP/HTTPS 服务器
2. SNMP Agent + 自定义 MIB
3. PPPoS 调制解调器
4. IPv6 双栈实现

## 代码阅读技巧

> [!tip] 阅读 lwIP 代码建议
> 1. **从入口点开始**: 如 `tcp_new()`, `pbuf_alloc()`
> 2. **理解数据结构**: PCB, pbuf, netconn
> 3. **跟踪调用链**: 使用 IDE 的查找引用功能
> 4. **配合抓包**: Wireshark 验证协议交互
