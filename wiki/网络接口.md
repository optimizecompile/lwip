# 网络接口 (netif)

## 概述

`netif` 是 lwIP 中网络接口的核心数据结构，代表一个网卡（物理或虚拟）。

## 源文件

- `src/core/netif.c` - netif 实现
- `include/lwip/netif.h` - 头文件

## netif 结构

```c
struct netif {
    struct netif *next;              // 下一个接口
    
    // IP 配置
    ip_addr_t ip_addr;               // IP 地址
    ip_addr_t netmask;               // 子网掩码
    ip_addr_t gw;                    // 网关
    
    // 输入/输出函数
    netif_input_fn input;             // 输入函数
    netif_output_fn output;           // 输出函数 (IP 层)
    netif_linkoutput_fn linkoutput;  // 链路层输出
    
    // 状态
    char name[2];                     // 接口名称 (e.g., "en")
    u8_t num;                        // 接口编号
    
    // 特性标志
    u16_t flags;
    
    // MTU
    u16_t mtu;
    
    // 硬件地址
    u8_t hwaddr_len;                 // 硬件地址长度
    u8_t hwaddr[NETIF_MAX_HWADDR_LEN];
    
    // 回调
    netif_status_callback_fn status_callback;
    netif_link_callback_fn link_callback;
};
```

## 注册网络接口

```c
#include "lwip/netif.h"

// 添加接口
err_t netif_add(struct netif *netif, 
                const ip_addr_t *ipaddr, const ip_addr_t *netmask,
                const ip_addr_t *gw, void *state,
                netif_init_fn init, netif_input_fn input);

// 设置默认接口
netif_set_default(netif);

// 启用/禁用接口
netif_set_up(netif);
netif_set_down(netif);
```

## 示例：以太网接口

```c
struct netif eth_netif;

static err_t netif_init_cb(struct netif *netif) {
    netif->mtu = 1500;
    netif->hwaddr_len = 6;
    
    // 设置 MAC 地址
    netif->hwaddr[0] = 0x00;
    netif->hwaddr[1] = 0x11;
    netif->hwaddr[2] = 0x22;
    netif->hwaddr[3] = 0x33;
    netif->hwaddr[4] = 0x44;
    netif->hwaddr[5] = 0x55;
    
    return ERR_OK;
}

void init_network(void) {
    ip_addr_t ip, mask, gw;
    
    IP4_ADDR(&gw, 192, 168, 1, 1);
    IP4_ADDR(&ip, 192, 168, 1, 100);
    IP4_ADDR(&mask, 255, 255, 255, 0);
    
    netif_add(&eth_netif, &ip, &mask, &gw, NULL, netif_init_cb, ethernet_input);
    
    netif_set_default(&eth_netif);
    netif_set_up(&eth_netif);
}
```
